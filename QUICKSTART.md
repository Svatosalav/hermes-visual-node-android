# 🚀 Быстрый старт — Hermes Visual Node v2.0

## Сборка APK

```bash
cd ~/hermes-visual-nodes/android-app
./build_v2.sh
```

APK будет в: `hermes-visual-node-v2.0-debug.apk`

## Установка

```bash
adb install hermes-visual-node-v2.0-debug.apk
```

Или через GitHub Actions (если настроено):
```bash
# Пуш в репозиторий автоматически соберёт APK
git push origin main
# Скачать из Releases
```

## Первый запуск

1. **Открыть приложение**
2. **Настройки** (⚙️ в правом верхнем углу)
3. **Ввести URL сервера**: `http://192.168.1.100:8766`
4. **Ввести ID устройства**: `my-phone`
5. **Сохранить и подключиться**

## Включить плавающую кнопку

В настройках → **Запустить** в секции "Плавающая кнопка"

При первом запуске система попросит разрешение "Отображение поверх других приложений" — разрешите.

## Включить полное управление (опционально)

Для команд клика, свайпа, ввода текста:

1. Настройки Android → **Специальные возможности**
2. Найти **Hermes Visual Node**
3. **Включить**
4. ⚠️ Подтвердить (это даст полный контроль!)

## Использование

### Чат в приложении
- Открыть приложение
- Написать сообщение
- Нажать 📷 для скриншота
- Отправить

### Плавающая кнопка
- Нажать на кнопку → скриншот
- Автоматически откроется overlay чат
- Добавить текст
- Отправить

### Команды Алисе

Примеры того, что можно попросить:

```
"Открой Chrome"
"Кликни на кнопку OK"
"Введи текст Hello World"
"Свайпни вниз"
"Нажми назад"
"Что написано на экране?"
"Найди кнопку Settings и кликни"
```

## Проверка работы

1. Запустить плавающую кнопку
2. Нажать на неё
3. Должен появиться overlay чат со скриншотом
4. Написать "Привет" и отправить
5. Алиса должна ответить

## Troubleshooting

**Плавающая кнопка не появляется**
→ Настройки → Приложения → Hermes Visual Node → Отображение поверх других приложений → Разрешить

**Команды не выполняются**
→ Настройки → Специальные возможности → Hermes Visual Node → Включить

**Не подключается к серверу**
→ Проверить URL, убедиться что телефон и сервер в одной сети
