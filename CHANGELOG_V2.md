# Hermes Visual Node v2.0 — Что нового?

## 🎨 Полностью переработанный интерфейс

### Было (v1.0):
- Простые XML layouts
- Базовая настройка через EditText
- Только плавающая кнопка

### Стало (v2.0):
- **Jetpack Compose** — современный UI
- **Material Design 3** — красивые цвета Яндекс Алисы
- **Встроенный чат** — полноценное общение с Алисой
- **Экран настроек** — удобное управление

## 💬 Встроенный чат с Алисой

- Отправка текстовых сообщений
- Прикрепление скриншотов
- История переписки
- Красивые пузыри сообщений
- Автоскролл к новым сообщениям

## 📸 Улучшенная работа со скриншотами

- Нажатие на плавающую кнопку → скриншот → автоматически открывается чат
- Превью скриншота перед отправкой
- Возможность добавить текст к скриншоту
- Удаление скриншота до отправки

## 🎯 Полное управление телефоном

Новый **AccessibilityService** позволяет Алисе:

- ✅ Кликать по координатам
- ✅ Делать свайпы
- ✅ Вводить текст в поля
- ✅ Нажимать Назад/Домой/Недавние
- ✅ Открывать уведомления
- ✅ Запускать приложения
- ✅ Искать элементы по тексту
- ✅ Кликать по тексту элемента
- ✅ Читать весь текст с экрана

## 🏗️ Архитектура

```
v1.0: Activity → Service → OkHttp
v2.0: Compose → ViewModel → Retrofit + AccessibilityService + PhoneController
```

## 📦 Новые зависимости

- Jetpack Compose (UI)
- Material 3 (дизайн)
- Retrofit (API)
- Coil (изображения)
- Coroutines (асинхронность)

## 🚀 Быстрый старт

```bash
# Сборка
cd ~/hermes-visual-nodes/android-app
./build_v2.sh

# Установка
adb install hermes-visual-node-v2.0-debug.apk

# Настройка
1. Открыть приложение
2. Настройки → ввести URL сервера
3. Включить плавающую кнопку
4. Включить AccessibilityService (для полного управления)
```

## ⚠️ Важные изменения

### Разрешения
Теперь требуется **AccessibilityService** для полного управления телефоном.
Это даёт приложению доступ ко всему на экране!

### API
Новые endpoints:
- `POST /chat` — отправка сообщений
- `GET /chat/history` — история чата
- `POST /command/result` — результаты команд

### Безопасность
AccessibilityService может:
- Читать всё на экране
- Выполнять любые действия
- Открывать любые приложения

**Используйте только в доверенной сети!**

## 🔄 Миграция с v1.0

1. Удалите старую версию (опционально)
2. Установите v2.0
3. Настройте URL сервера заново
4. Включите AccessibilityService
5. Готово!

Настройки сохраняются в SharedPreferences, но лучше ввести заново.

## 🐛 Известные проблемы

- AccessibilityService может конфликтовать с другими accessibility приложениями
- На некоторых устройствах overlay чат может не работать (ограничения производителя)
- Скриншоты требуют разрешение при каждом запуске (Android ограничение)
